# Hoopcharts - frontend

## Dependencies
- GraphQL (appolo client)
- Vue 3
- Hosting on vercel

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
